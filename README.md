# Chat-Bot

Este repositorio contiene el desarrollo de un chatbot implementado en **Python**, diseñado para interactuar de manera inteligente con usuarios y proporcionar respuestas automatizadas. El proyecto está pensado para facilitar la integración y el despliegue en diferentes entornos, permitiendo ampliar sus funcionalidades de acuerdo con las necesidades específicas.

## Tecnologías usadas

- **Python** (100%): lenguaje principal de desarrollo.

## Modo de despliegue

Para desplegar el chatbot, sigue estos pasos básicos:

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/Json-Esutpinan/Chat-Bot.git
   cd Chat-Bot
   ```

2. **Instala las dependencias:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Ejecuta la aplicación:**
   ```bash
   uvicorn app.main:app
   ```
